## 💡 Pro Tips for Users

1. **More data = better forecasts**
   - 7 days = minimum
   - 30 days = good
   - 60+ days = best

2. **Use realistic numbers**
   - Your actual storage data
   - Approximate is OK for testing

3. **One device at a time**
   - One user_id per upload
   - Upload multiple times for multiple devices

4. **Keep template structure**
   - Don't delete columns
   - Don't change column names
   - Just fill the values
