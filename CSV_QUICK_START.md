# ⚡ 30-Second CSV Guide

## TL;DR - The Fastest Way

1. Open the **Data Builder** tab in the app
2. Click **"🔨 Generate Template"**
3. Fill in your device info:
   - Device ID (e.g., `my_phone`)
   - Number of days (7-30)
   - Storage capacity (GB)
   - Start date
4. Click **"Generate blank template"** → Download CSV
5. Open in Excel/Sheets → Fill in your numbers
6. Go to **Forecast Lab** → Upload CSV → Get forecast!

---

## What Numbers Do I Need?

Every day, record:
- **used_gb**: How much storage you're using
- **free_gb**: How much is left (usually = capacity - used)
- **Photos/Videos/Apps/Documents/System/Other**: Storage by type

That's it! The template handles the rest.

---

## Where's My Data?

### On iPhone:
Settings → General → Storage → See what's using space

### On Android:
Settings → Storage → See breakdown

### On Computer:
Disk Management (Windows) or Disk Utility (Mac)

---

## Common Questions

**Q: Do I need exactly 30 days?**  
A: No, minimum 7 days. More data = better forecasts.

**Q: What if I only have 5 days?**  
A: Use **7 days minimum**. The app needs a week to see patterns.

**Q: Can I use data from last year?**  
A: No, recent data works best (last 30 days ideal).

**Q: Do the numbers need to be exact?**  
A: Yes, but approximate is OK if you're testing.
