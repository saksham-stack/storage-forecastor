## 🚀 How to Use It

### Fastest Way (2 minutes)
```
1. streamlit run dashboard\app.py
2. Click "Data Builder" tab
3. Click "Download Real Example"
4. Go to "Forecast Lab" tab
5. Upload the CSV
6. Get forecast! 🎉
```

### Custom Data Way (10 minutes)
```
1. streamlit run dashboard\app.py
2. Click "Data Builder" tab
3. Click "Generate Template"
4. Fill 5 simple fields
5. Download blank CSV
6. Open in Excel/Sheets
7. Fill your storage numbers
8. Go to "Forecast Lab" → Upload → Forecast! 🎉
```

### Learn First Way (20 minutes)
```

6. Download → Fill → Upload → Forecast! 🎉
```
