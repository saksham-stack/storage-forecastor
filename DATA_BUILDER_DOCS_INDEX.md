```

   streamlit run dashboard\app.py

3. Choose method:

   - "Generate Template" (create your CSV)
```

## ✅ Checklist for Users

Before using Data Builder:
- [ ] Dashboard is running (`streamlit run dashboard\app.py`)
- [ ] You found the "Data Builder" tab
- [ ] You know which method to use
- [ ] You have storage data (for Method 2)

After using Data Builder:
- [ ] You downloaded a CSV
- [ ] You have 7+ days of data
- [ ] All 16 columns are filled
- [ ] No negative numbers
- [ ] Dates in YYYY-MM-DD format

Ready to forecast:
- [ ] Go to "Forecast Lab" tab
- [ ] Choose "Upload CSV"
- [ ] Upload your file
- [ ] See forecast! 🎉
