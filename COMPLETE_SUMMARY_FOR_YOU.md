## Quick Start

### Fastest Way (2 minutes)
```bash
# 1. Run the app
streamlit run dashboard\app.py

# 2. Click "Data Builder" tab

# 3. Click "Download Real Example"

# 4. Go to "Forecast Lab" tab

# 5. Upload the CSV

# 6. Get your forecast! 🎉
```

### Custom Data Way (10 minutes)
```bash
# 1. streamlit run dashboard\app.py

# 3. Fill form (5 fields)
# 4. Download blank CSV
# 5. Open in Excel/Sheets
# 6. Fill your storage numbers
```
